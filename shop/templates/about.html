About This Project

Welcome to my e-commerce website — a personal project built using Django. This site simulates the core functionality of a modern online store and serves as a portfolio project to showcase my web development skills.

🔧 Key Features

  • User authentication (signup, login, logout)
  • Product listing and detailed product pages
  • Shopping cart and order management
  • Admin dashboard for product control
  • Search feature

🛠️ Tech Stack

  • Backend: Django, SQLite
  • Frontend: HTML5, CSS3, Bootstrap
  • Tools: Git, GitHub, VS Code
